Bootstrap Project

Responsive Portfolio

Overview

This is a sample landing page with responsive design built using Bootstrap frameworks. The gallery layout is ideal for those businesses who rely heavily on images.

Objectives

  • Organize and highlight portfolio content
  • Design and develop fully responsive solutions

Requirements

  • UX design
  • Responsive Design
  • Front End Development
  • Bootstrap Frameworks

Design Process

Before beginning the project, I created a user persona - a description of an ideal person who would need a site like this and then built the website based on the profile of this individual. I chose a thumbnail gallery because my client's content is image-focused and it is a necessity that it shines above all else.

Results

After creating this page, I also built an additional project page to showcases each photo up-close with technical specs, notes from the photographer, price and how to purchase. While this site is not currently launched, you can preview it upon request.
