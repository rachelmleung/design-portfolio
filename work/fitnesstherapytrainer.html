Fitness Therapy Trainer

Responsive Re-Design

Overview

The goal of this project was to lighten up the page for easier reading and make sure the design would be mobile and tablet friendly.

Objectives

  • Improve readibility and ease of use
  • Organize portfolio content
  • Design responsive solutions

Requirements

  • UX design
  • Responsive Design
  • SEO

Design Process

We decided readibility and ease of use were the two most essential goals for this website resdesign. His target market prefers simple designs that focus on his knowledge and expertise. It was essential that these new designs were also made to work on phones and tablets.

( Previous Web Design - 2016 )

First I focused on creating a color palette and gathering font options that would brighten the look of the website and make it seem more inviting and helpful. He wanted to keep the black & white photos so it was essential that we focused on making the page pop with a little color. Next we decided more obvious links and re-distributed images would be the best place to improve usability.

( Style Tile for Fitness Therapy Trainer )

Results

The site design is now more appealing to both the target market and new visitors. It's much more user friendly now thanks to simple changes like colors, links, fonts and reorganizing the same content.
